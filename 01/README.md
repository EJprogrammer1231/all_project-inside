# 💻 Smart To-Do List Project Challenge

Build a **Smart Task Manager** using **HTML, CSS, and JavaScript**.

The goal of this project is to practice:

* DOM manipulation
* Arrays and objects
* JavaScript algorithms
* Basic UI development

---

# 🎯 Project Features

## 1️⃣ Add Task | **DONE**

The user can type a task in an input field and press **Add**.

Example tasks:

* Study JavaScript
* Practice DSA
* Build a project

---

## 2️⃣ Display Tasks **DONE**

All tasks must appear inside a **list (`ul`)**.

Example display:

• Study JavaScript
• Practice DSA
• Build a project

---

## 3️⃣ Mark Task as Completed **----- CURRENTLY WORKING HERE -----**

When the user clicks a task:

* The task should be marked **completed**
* The text should appear **crossed out**

Example:

Study JavaScript ✔

---

## 4️⃣ Delete a Task

Each task must have a **Delete button**.

Example:

Study JavaScript ❌

When clicked, the task should be **removed from the list**.

---

## 5️⃣ Show Task Statistics

Your app must display task statistics:

Total Tasks: 5
Completed Tasks: 2
Remaining Tasks: 3

These values must update **automatically when tasks change**.

---

# 📦 Data Structure Requirement

Store tasks in an **array of objects**.

Example structure:

tasks = [
{ text: "Study JS", completed: false },
{ text: "Build project", completed: true }
]

---

# 🧠 Algorithms You Should Use

You should practice using:

push() → add a task
forEach() → display tasks
filter() → count completed tasks
length → calculate totals
